public-dgk(DGK; Dc Gonic Killer의 공개판)는 인터넷 커뮤니티 사이트 DCinside에서 특정한 고닉 유저의 게시글을 원하는 만큼 자동으로 올려주는 간단한 프로그램이다.
C언어를 공부하면서, 뭔가 진짜로 써먹을 수 있는 프로그램을 직접 만들어 보자는 생각으로 만들었다.
형편없는 실력에, 코드도 스파게티보다 더 꼬인 코드이고 Linux 환경에서밖에 돌아가지 않는 엉터리 코드이지만 100줄 넘는 코드는 처음 짜봐서 GitHub에 남겨두고 싶어서 올린다.

프로그램은 dgk와 streplace로 구성되어 있다.
dgk는 입력한 이용자의 ID와 갤러리 주소, 기준 게시글 번호, 올릴 조회수를 입력하면, 기준 게시글 번호를 시작으로 가장 최근에 올라온 게시글까지 조회하면서 해당 유저의 게시글이 올라오는 족족 게시글의 조회수를 증폭시킨다.
아카이브 기능을 켜면 게시글을 조회할 때 저장한 HTML 파일을 저장한다.

streplace는 dgk에서 아카이브 기능을 사용할 때, DCinside 이미지 서버에서 외부 이미지 조회를 막아놓은 것을 우회하도록 HTML 코드를 변경해주는 프로그램이다.
